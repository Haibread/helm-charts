# helm-charts

A collection of Helm charts for self-hosted apps.

## Charts

| Chart | Description | Version |
| ----- | ----------- | ------- |
| [suggestarr](charts/suggestarr) | Media-recommendation engine for *arr stacks | 0.1.0 |
| [radarr](charts/radarr) | Movie collection manager | 0.1.0 |

## Usage

```sh
helm install my-release ./charts/<chart-name>
```

Each chart's `README.md` documents the values it accepts.
